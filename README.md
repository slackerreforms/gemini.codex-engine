# Codex Engine

**A powerful single-file AI writing companion for novelists.**

An advanced browser-based workspace with deep world-building tools and Gemini-powered AI features.

### Features
- World Bible + Active Mysteries
- Full Character, Location, and Organization managers
- Chapter Planner
- Character Interview (talk to your characters)
- Consistency Engine (detects plot holes)
- Story Architect, Rephraser, Foreshadowing, Timeline, etc.
- Everything saved locally in your browser

### How to Use
1. Download or open `index.html` in any browser
2. Enter your **Gemini API key** (free from [Google AI Studio](https://aistudio.google.com/apikey))
3. Start building your novel's canon

**No installation. No accounts. Works completely offline after first load.**
